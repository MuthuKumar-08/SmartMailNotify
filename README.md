# SmartMailNotify# 📧 SmartMailNotify

AI-powered Gmail automation system built with Python.

## Features

- Gmail OAuth Authentication
- Read unread Gmail messages
- SQLite Database
- Duplicate Detection
- Modular Architecture
- AI-ready Design
- WhatsApp Integration (Coming Soon)

- SQLite

- VS Code
